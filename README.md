# 🚀 Formulário de Cadastro de Veículos


📌 Sobre o projeto

Este projeto é uma aplicação web que disponibiliza um formulário para cadastro de veículos, permitindo que o usuário insira informações detalhadas sobre o carro, como preço, marca, modelo, quilometragem, opcionais e fotos.  
Foi desenvolvido para fins educacionais e de portfólio.

________________________________________
🎯 Objetivo  
Facilitar o processo de cadastro de veículos para venda, oferecendo um formulário completo e intuitivo.  

________________________________________
✨ Funcionalidades  
- Cadastro de título, preço e descrição do veículo  
- Inserção de marca, modelo e quilometragem  
- Seleção da data de compra  
- Escolha do tipo de câmbio (manual ou automático)  
- Seleção de opcionais como airbag, ar-condicionado, câmera de ré, bancos de couro, entre outros  
- Upload de múltiplas imagens do veículo  
- Validação de campos obrigatórios  

________________________________________
🛠️ Tecnologias utilizadas  
- **HTML5** para estruturação da página  
- **CSS3** para estilização e layout responsivo
